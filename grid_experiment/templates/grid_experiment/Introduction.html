Task Overview

In this study, you will complete a series of rounds of a perceptual identification task. Each round follows the same sequence:

  1. View the grid: A {{ grid_size }} grid filled with symbols X and O will be displayed for {{ display_seconds }} seconds.
  2. Identify the scenario: Report which symbol appeared more frequently. Correct identification earns you {{ identification_bonus }}.
  3. Choose an action: Select between Action A or Action B. Your payoff depends on which scenario is active.

Viewing Options

Before submitting your identification, you may request to view the grid again. {% if view_again_cost > 0 %} Each additional viewing costs {{ view_again_cost }}. {% else %} Additional viewings are free, but the number of requests will be recorded. {% endif %}

Scenarios

There are two possible scenarios that determine the payoffs: {{ scenario_label_moral }} and {{ scenario_label_win }}. The symbol that appears more frequently in the grid indicates which scenario is active for that round.

On the next page, you will learn which symbol corresponds to which scenario, along with the payoff structure for each scenario.

Please work carefully and make the best decisions you can on each round. Your decisions affect both your own payoff and that of a charity.
